# rgesntracker
A simple tool for tracking and displaying the current status of the [RGESN](https://ecoresponsable.numerique.gouv.fr/publications/referentiel-general-ecoconception/) network.

## Usage
```bash
# Install dependencies
$ npm install

# Run the program
$ npm true start
```
